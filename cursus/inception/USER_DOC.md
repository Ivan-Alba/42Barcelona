# User Documentation - Inception Stack

## Services Overview
- **Web Server (Nginx)**: Secure entry point via HTTPS (Port 443).
- **WordPress**: The main CMS running on PHP-FPM.
- **MariaDB**: Relational database for content storage.
- **Redis Cache**: High-performance object cache for WordPress.
- **Adminer**: Web-based database management tool.
- **FTP Server**: Secure file access to the WordPress volume (Port 21).
- **Static Site**: A secondary portfolio site (Port 8080).
- **Portainer**: Visual dashboard for monitoring and managing containers.

## Operational Guide
- **Starting the project**: Execute `make` in the root directory.
- **Stopping the project**: Execute `make stop`.
- **System Health**: Run `docker ps` to ensure all 8 services (including bonuses) are "Up".

## Credential Management
Credentials are automatically generated by the `Makefile` and are **not** stored in the repository. To find your passwords, check the following local files:
- `secrets/db_root_password.txt`
- `secrets/db_password.txt`
- `secrets/ftp_password.txt`
